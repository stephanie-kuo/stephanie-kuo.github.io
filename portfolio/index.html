REMINDer

REMINDer is an application designed for people with ADD/ADHD. Considering to their characteristic of less concentrations, we designed a task tracking tool to help them focus on current task and achieve their own goals.

LinkTogether

LinkTogether is a social network application for organizations with flat corporate structure to improve the communication within the organization. With LinkTogether, employees can identify the key person easily and enhance the connection between each other.

MomKit

MomKit is a mobile health app that teaches parents evidence-based breastfeeding techniques with Pixar-style animations. With MomKit, mothers can learn how to alleviate, improve, or prevent the most common breastfeeding problems.

Millions of Memories is a game application on tablet, it can help people better understand their families and create delightful family gathering time.
